# Personal Finance Tracker (CLI) 💰

RU: Консольное Java-приложение для управления личными финансами: учет транзакций, 
аналитика расходов и управление бюджетом.


EN: A CLI-based Java application for personal finance management: transaction tracking, 
expense analytics, and budget control.

---

## 🛠 О проекте
Данное приложение предназначено для ведения учёта личных доходов и расходов через консольный интерфейс. Проект разработан с использованием принципов **ООП** и модульной архитектуры, что обеспечивает чистоту кода и лёгкость расширения функционала.

## 🚀 Основные возможности
- **Учёт транзакций:** Добавление, удаление и просмотр доходов и расходов.
- **Категоризация:** Группировка трат по категориям (Еда, Транспорт, Жильё и т.д.).
- **Бюджетирование:** Установка лимитов на месяц и уведомления об их превышении.
- **Аналитика:** Генерация отчётов по категориям с визуализацией в консоли.
- **Persistence:** Автоматическое сохранение данных в файл и загрузка при старте.
- **Экспорт:** Возможность выгрузки данных в формат CSV (для Excel).


## 🛠 Технологический стек
- **Язык:** Java 17+
- **Архитектура:** Модульная (Separation of Concerns)
- **Хранение данных:** Работа с файловой системой (I/O)
- **Управление версиями:** Git + GitHub
